# Nepali Calendar Engine

TypeScript package for Bikram Sambat (BS) calendar conversion, panchang lookup, and event resolution.

## What you can do

- Convert BS dates to AD and AD dates to BS.
- Build month views with rich day metadata.
- Read panchang values (tithi, paksha, nakshatra, yoga, karana).
- Resolve festivals and auspicious date signals.
- Validate data with offline-first trust checks and cross-validation workflows.
